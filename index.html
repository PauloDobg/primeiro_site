Hospedagem GitHub

Para criar um site no GitHub.com devemos ter uma conta válida e um repositório com o arquivo index.html.

Hospedando um Site

  1. Criar um repositório
  2. Publicar o repositório utilizando o comando Publish
  3. Abrir o repositório no VSCode
  4. Criar o index.html
  5. Criar a versão usando o comando commit
  6. Enviar para o repositório remoto usando o comando Push
  7. Entrar no GitHub.com, acessar o repositório, ir na opção --> Settings --> Pages --> Selecionar Branch
  8. Clicar na URL gerada.

Branches

São ramificações do seu projeto.

O projeto principal em geral fica na branch main. A branch main deve ter a versão final e funcional do seu projeto.
O projeto pode ter Branches secundárias.
